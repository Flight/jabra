## Quick start for developers
Project uses node + angular 2 + Gulp for fast developing.

To install node: https://nodejs.org/en/

Then open your favorite Terminal and run these commands.

```sh
$ npm install -g gulp
$ npm install
```

To build project:
```sh
$ gulp build
```

To run server and watch changes:
```sh
$ gulp serve
```
